<h1>Sistema de Controle de Compras</h1>

Uma programa em Java que simula o uso de um cartão de crédito, permitindo registrar compras, controlar o saldo disponível e exibir um histórico organizado das transações realizadas. O programa utiliza princípios de orientação a objetos para separar responsabilidades e garantir uma estrutura clara.

- **Funcionalidades:**
  - **Cadastro de compras:** O usuário informa a descrição e o valor da compra, que então é registrada como um objeto independente.
  - **Controle de limite e saldo:** O sistema valida automaticamente se há saldo disponível antes de autorizar uma nova compra.
  - **Atualização dinâmica do saldo:** Cada compra aprovada reduz o valor disponível no cartão.
  - **Listagem ordenada de compras:** Todas as compras realizadas são exibidas, organizadas pelo valor com a implementação da interface Comparable.
  - **Interação direta via console:** O usuário insere dados, confirma ações e acompanha as mensagens do sistema em tempo real.
  - **Uso de coleções:** As compras são armazenadas em uma lista, facilitando ordenações, consultas e manipulação dos dados.
